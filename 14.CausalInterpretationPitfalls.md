# Correlation ≠ Causation: 정형 데이터 해석의 인과 함정

> SHAP·변수중요도·회귀계수를 **인과효과로 오해석**할 때 생기는 문제와 그 정식 명칭, 대표 의료 사례, 안전장치를 정리한다.
> 핵심 한 줄: **SHAP는 "모델"을 설명할 뿐 "세계(인과 구조)"를 설명하지 않는다.** 예측에 유용한 변수가 곧 원인은 아니다.

## 1. 문제 정의

정형 데이터에 XGBoost·로지스틱 회귀 등을 학습시키고 SHAP/변수중요도/회귀계수를 뽑으면 "어떤 변수가 결과에 중요한가"가 나온다. 문제는 이 산출물이 **연관(association)·예측(prediction)** 의 척도이지 **인과효과(causal effect)** 의 척도가 아니라는 점이다. 그럼에도 실무에서는 "X의 SHAP가 크다 → X가 Y의 원인이다 → X를 바꾸면 Y가 바뀐다"로 비약하기 쉽다.

이 비약은 논리학에서 **상관-인과 오류(correlation–causation fallacy, *cum hoc ergo propter hoc*)** 로 불리고, 역학(疫學)에서는 회귀표의 보조 변수 계수를 전부 인과로 읽는 특정 형태에 **Table 2 fallacy(표 2의 오류)** 라는 고유 명칭까지 붙어 있다 [1]. 즉 이것은 막연한 주의사항이 아니라 **이미 정식으로 정의된 문제**다.

의료에서 특히 위험한 이유는, 통계적으로 그럴듯한 연관이 **의학 지식과 정면으로 모순**되는 결론으로 이어질 수 있기 때문이다. "천식이 폐렴 사망을 낮춘다", "호르몬제가 심장을 보호한다", "흡연이 코로나를 막는다" 같은 결론은 모두 실제 데이터에서 **통계적으로 관측된** 연관이지만, 인과로 읽으면 환자를 해칠 수 있다.

## 2. 핵심 명칭 총정리

| 명칭 (영문 / 국문) | 정의 | SHAP·회귀에서 나타나는 양상 |
|---|---|---|
| **Confounding / Confounder** (교란 / 교란변수) | 노출 X와 결과 Y를 동시에 일으키는 제3변수 Z. DAG로 `X ← Z → Y` | X가 Z의 신호를 대신 받아 큰 계수/SHAP를 가짐. Z를 보정하면 X 효과가 줄거나 뒤집힘 |
| **Spurious correlation** (허위 상관) | 인과 연결 없이 교란·우연·선택으로 생긴 연관 | 인과 경로가 없는 변수가 높은 중요도를 가짐. 진짜 원인을 넣으면 사라짐 |
| **Correlation–causation fallacy** (상관-인과 오류) | "상관 → 인과"로 추론하는 핵심 오류 | 0이 아닌 SHAP/계수를 곧바로 "X가 Y를 일으킨다"로 읽음 |
| **Table 2 fallacy** (표 2의 오류) | 한 회귀모델의 *보조 변수* 계수들을 각각 타당한 인과효과로 해석 [1] | 변수중요도 순위·계수표를 "원인 목록"으로 읽음 — 잔여 교란·매개 보정 탓에 무효 |
| **Simpson's paradox** (심슨의 역설) | 통합 데이터의 연관 방향이 모든 하위그룹에서 반대로 나타남 | 풀링 데이터에서의 부호가 층화 시 뒤집힘 |
| **Collider bias / Berkson's paradox** (충돌부 편향 / 버크슨의 역설) | 공통 결과(collider, `X → C ← Y`)를 조건화하면 가짜 X–Y 연관이 생김 | 선택된 표본(입원자·검사자만)에서 없던 연관(흔히 "보호 효과")이 생김. 하류 collider를 feature로 넣으면 안 됨 |
| **Selection bias** (선택 편향) | 표본이 collider 기준으로 정의되어 생기는 일반적 편향 | 코호트 구성 자체가 변수–결과 관계를 왜곡 |
| **Reverse causation** (역인과) | 실제로는 `Y → X`인데 모델이 `X → Y`로 읽음 | 질병이 *낮춘* 바이탈/바이오마커가 강한 "위험인자"로 보임 |
| **Confounding by indication** (적응증에 의한 교란) | 약을 처방하는 임상적 이유(중증도)가 결과에 독립적으로 영향 | "치료" 변수가 해로워 보이거나(중증 환자에 투여), 예방약이 마법처럼 보호적으로 보임 |
| **Healthy-user / adherer bias** (건강사용자/건강순응자 편향) | 예방요법을 택하거나 잘 따르는 사람이 체계적으로 더 건강함 | 복약·순응 변수가 미측정 건강행동을 대리해 과대 효과 |
| **Immortal time bias** (불멸시간 편향) | 정의상 결과가 발생할 수 없는 추적 구간을 노출군에 잘못 귀속 | 치료로 정의된 변수가 가짜 생존 이득을 가짐 |
| **Ecological fallacy** (생태학적 오류) | 집단/지역 수준 관계를 개인 수준으로 추론 | 지역·집단 평균 변수의 중요도를 개인 특성으로 오해 |
| **Proxy / surrogate variable** (대리변수) | 미측정 진짜 원인과 상관된 측정 가능한 대체물 | 대리변수의 높은 SHAP는 숨은 원인의 효과(예: 우편번호 = 흡연·소득·대기오염) |
| **M-bias** (M-편향) | 두 미측정 원인의 공통 결과인 사전변수를 보정하면 오히려 편향 경로가 *열림* | "변수를 더 통제"하거나 feature를 추가하는 것이 편향을 *유발* |

> 정리하면, "강남 살면 폐암" 같은 오류는 **교란 + 대리변수 + 생태학적 오류**가 겹친 경우이고, 혈압 사례는 대개 **역인과**나 **적응증 교란**으로 설명된다.

## 3. "강남 살면 폐암" 메커니즘

변수: **X = 거주지(강남)**, **S = 흡연(진짜 원인)**, **Y = 폐질환**.

- 실제 인과 구조: **`S → Y`** (흡연이 폐질환을 일으킴). 거주지는 폐질환을 직접 일으키지 않으므로 **`X → Y` 화살표가 없다.**
- 그런데 거주지는 흡연과 상관된다(인구·소득·문화로 지역별 흡연율이 다름): **`X ↔ S`** (X는 S의 대리/표지자).
- 모델이 S를 못 쓰거나 약하게 쓰면, **X가 S의 정보를 담고 있으므로** X가 Y를 잘 예측한다. SHAP는 **X에 큰 값**을 주지만, 이는 S의 인과 신호를 빌린 것이다. 이는 동시에:
  - **교란/대리(proxy)**: X가 미측정(또는 동시측정) 진짜 원인 S를 대신함.
  - **생태학적(ecological)**: X는 지역 집계량이라 흡연 신호가 개인 속성이 아닌 집단 평균임.

**핵심 검정 — 조건부 독립:** **`X ⫫ Y | S`**. 흡연을 조건화하면 거주지는 폐질환에 대해 더 이상 정보를 주지 않는다. 즉 S를 보정하면 거주지의 SHAP는 ~0으로 붕괴한다. 이 조건부 독립이 "거주지 중요도는 순전히 흡연 대리였다"는 신호다. 거주지를 **개입(다른 동네로 이사)** 해도 폐질환 위험은 안 바뀌지만, 흡연을 개입하면 바뀐다 — 예측과 인과의 차이가 여기서 갈린다.

## 4. 대표 의료 사례

각 사례: **(잘못된 연관 해석) → (실제 인과) → (해당 편향) → (검증된 인용)**.

### 4.1 호르몬 대체요법(HRT)과 심장병 — 관찰연구 "보호" vs RCT 반전
- **잘못된 해석**: 폐경 후 에스트로겐/HRT 사용자가 관상동맥질환(CHD)이 ~35–50% *낮음* → "HRT가 심장을 보호한다" [2].
- **실제**: 건강사용자 편향. HRT 사용자가 더 건강·고소득·건강지향적이었다. 무작위 시험(WHI)이 이를 제거하자 E+P는 CHD를 예방하지 *못했고* 초기엔 위험을 *높였다* [3].
- **편향**: 적응증/건강사용자 교란.
- 관찰 주장: Grodstein et al., *NEJM* 1996 (Nurses' Health Study) [2]. RCT 반전: Writing Group for the WHI (Rossouw et al.), *JAMA* 2002 [3].

### 4.2 폐렴 위험 모델 — "천식이 사망을 낮춘다" (Caruana et al. 2015)
- **잘못된 해석**: 천식 있는 폐렴 환자의 사망률이 *더 낮음* → "천식이 보호적".
- **실제**: 천식 폐렴 환자는 곧장 중환자실/적극 치료로 분류되어 사망률이 낮아진 것. 모델은 *치료 정책·선택* 효과를 학습했을 뿐 생물학이 아니다. 규칙대로 천식 환자를 귀가시키면 치명적이다.
- **편향**: 적응증 교란 + 예측≠인과. 해석 가능한 모델(GA²M)이 이 위험한 규칙을 노출시켜 제거할 수 있었다.
- 인용: Caruana et al., *KDD '15* [4].

### 4.3 항부정맥제 — 가정된 이득의 RCT 반전 (CAST)
- **잘못된 해석**: 심근경색 후 조기심실수축을 항부정맥제(encainide/flecainide)로 억제하면 이로울 것(병태생리·관찰적 추론).
- **실제**: 무작위 CAST 시험에서 이 약들이 오히려 사망을 *증가*(치명적 부정맥사). 적응증 교란된 약–결과 연관은 신뢰할 수 없고 RCT가 필요함을 보여준 고전.
- **편향**: 적응증/중증도 교란.
- 인용: Echt et al. (CAST), *NEJM* 1991 [5].

### 4.4 역인과 — 낮은 콜레스테롤/혈압과 사망
- **잘못된 해석**: 낮은 총콜레스테롤(또는 낮은 혈압·체중)이 높은 사망률과 연관 → "낮은 콜레스테롤이 위험하다".
- **실제**: 잠복/초기 질환(암·만성질환·노쇠)이 사망 전에 콜레스테롤·혈압·체중을 *낮춘다*. 바이오마커는 질병의 *결과*이지 사망의 원인이 아니다.
- **편향**: 역인과.
- 인용(맥락): Jacobs et al., *Circulation* 1992 (저콜레스테롤–사망 연관 회의) [6]. → 자세한 혈압 논의는 §5.

### 4.5 심슨의 역설 — 신장결석 치료
- **잘못된 해석**: 통합 성공률에서 경피적 신쇄석술(83%)이 개복수술(78%)보다 우수 → "경피술이 낫다".
- **실제**: 개복수술이 *큰 결석*(성공률 낮은 범주)에 더 많이 쓰였다. 결석 크기로 층화하면 *각 층 모두에서* 개복수술 성공률이 더 높다. 결석 크기가 잠복변수라 통합 비교가 뒤집힌다.
- **편향**: 심슨의 역설 / 중증도 교란.
- 인용: Charig et al., *BMJ* 1986 [7]; 교육적 해설 Julious & Mullee, *BMJ* 1994 [8].

### 4.6 충돌부/선택 편향 — "흡연이 코로나를 막는다"와 출생체중 역설
- **잘못된 해석(코로나)**: 검사·입원 환자 중 흡연자의 코로나 양성/중증이 *낮음* → "흡연이 보호적".
- **실제**: 검사/입원(흡연 증상과 코로나 양쪽에 의해 결정되는 collider)을 조건화하면 선택 표본에서 가짜 음의 연관이 생긴다. 편향 없는 근거(혈청유병률 등)는 보호를 지지하지 않는다.
- 인용: Griffith et al., *Nature Communications* 2020 [9].
- **출생체중 역설(고전적 collider)**: 저체중아 중 흡연모 아기의 사망률이 *낮음* → "흡연이 저체중아를 보호". 거짓이다. 출생체중(흡연 S와, 사망을 높이는 미측정 원인 U의 공통 결과; `S → BW ← U → Y`)을 조건화하면 가짜 경로가 열린다.
- 인용: Hernández-Díaz et al., *Am J Epidemiol* 2006 [10].

### 4.7 불멸시간 편향 — 흡입 스테로이드와 COPD 생존
- **잘못된 해석**: 흡입 코르티코스테로이드(ICS)를 쓴 COPD 환자의 사망률이 낮음 → "ICS가 생존을 늘린다".
- **실제**: 퇴원 후 ICS 처방을 받아 "사용자"로 분류되려면 그때까지 *생존*해야 했다. 이 처방 전 "불멸" 생존시간이 약 효과로 오귀속되어 가짜 생존 이득이 생긴다. 시간의존 분석으로 보정하면 사라진다.
- **편향**: 불멸시간 편향.
- 인용: Suissa, *Eur Respir J* 2004 [11]; 방법론 앵커 Suissa, *Am J Epidemiol* 2008 [12].

### 4.8 비만 역설 — collider + 역인과
- **잘못된 해석**: 심혈관질환/심부전 *환자 중에서는* 비만이 더 *낮은* 사망률과 연관 → "비만이 아픈 환자를 보호".
- **실제**: 이미 질병이 있는 사람으로 제한(질병 = 비만과 미측정 위험인자 U의 공통 결과인 collider). 그 층에 든 비만 환자는 *다른 위험인자가 적은* 경향이라 비만이 보호적으로 보인다. 마른 사람의 질병성 체중감소(역인과)가 겹친다.
- **편향**: 충돌부/선택 편향 + 역인과.
- 인용: Banack & Kaufman, *Prev Med* 2014 [13] (반론 Sperrin et al., *Epidemiology* 2016 [14]).

## 5. 혈압·바이탈 사례 집중

혈압은 인과 함정이 특히 풍부한 변수다. "혈압이 낮을수록/약을 먹을수록 결과가 나쁘다"는 연관은 거의 항상 **역인과**나 **적응증 교란**으로 설명된다.

- **역인과 (낮은 혈압 → 사망?)**: 심부전·중증 질환·노쇠 환자에서 낮은 수축기 혈압이 높은 사망률과 연관된다. 이는 낮은 혈압이 사망을 일으켜서가 아니라, **망가진 심장·전신 쇠약이 혈압을 낮추기 때문**이다. 혈압은 중증도의 *결과*다. SHAP에서 "낮은 혈압"이 강한 위험인자로 떠도, 혈압을 인위로 올린다고 생존이 좋아지지 않는다. (혈압 치료의 이른바 **J-curve**: 지나친 강하가 해롭다는 관찰 연관도 상당 부분 기저 질환의 역인과·교란으로 설명된다.)

- **적응증에 의한 교란 (혈압약 → 나쁜 결과?)**: 순수 관찰 데이터에서 "항고혈압제 복용"이 심혈관 사건·낙상·골절과 양의 연관을 보일 수 있다. 이유는 약이 해로워서가 아니라 **고혈압·동반질환·노쇠가 있는 사람에게 처방되기 때문**(처방 이유 = 결과의 위험요인). "치료" 변수가 곧 "누가 아픈가"를 인코딩한다.

- **대리변수/생태학**: 지역 평균 나트륨 섭취와 혈압의 생태학적 상관을 개인 인과로 읽으면 생태학적 오류가 된다.

> 실무 함의: 혈압·심박·체중·콜레스테롤 등 **질병에 의해 변하는 바이탈/바이오마커**가 모델에서 강한 "위험인자"로 잡히면, 인과 결론 전에 반드시 **역인과 가능성**을 먼저 의심해야 한다. 또 "치료/약물" 변수의 중요도는 적응증 교란 때문에 **인과효과로 읽으면 안 된다** — 이 질문은 RCT 또는 target trial emulation으로만 답할 수 있다.

## 6. ML/SHAP 특화 쟁점

### 6.1 모델 설명 vs 세계 설명
SHAP·변수중요도는 **"학습된 함수 f가 입력을 어떻게 쓰는가"**(f와 데이터 분포의 성질)를 설명하지, **"현실에서 어떤 변수에 개입하면 Y가 바뀌는가"**(인과효과)를 설명하지 않는다. 예측에 유용한 변수가 인과적일 필요는 없다 — 대리변수·collider·매개변수일 수 있다.

### 6.2 Interventional vs Conditional(Observational) Shapley
- **Interventional / marginal Shapley** ("true to the model"): 상관을 무시하고 변수를 독립적으로 교란 → 모델이 각 변수에 *의존하는 정도*를 분리. 모델에 대한 do()-스타일 읽기에 가깝다.
- **Conditional / observational Shapley** ("true to the data"): 변수 간 상관을 존중 → 상관된 변수들에 신용을 분산. 데이터 분포는 더 잘 묘사하지만 **대리/비원인 변수에도 중요도를 배분**한다.
- 두 변형 중 무엇도 모델 자체가 올바른 인과 모델이 아닌 한 **현실의 인과효과와 같지 않다** [15]. Shapley 값에서 인과적 의미를 끌어내려면 설명 기법만으로는 부족하고 **명시적 인과 가정(DAG)** 이 필요하다 [15][16].

### 6.3 정리
SHAP가 인과적으로 타당하려면 **(a)** 기저 모델 자체가 올바른 인과 모델이고(교란변수 보정, collider를 feature로 넣지 않음) **(b)** 목적에 맞는 Shapley 변형을 써야 한다. 기본값은 "모델 설명"이지 "세계 설명"이 아니다.

## 7. 실무 안전장치

| 장치 | 내용 |
|---|---|
| **인과 그래프(DAG)** | 모델링 *전에* 가정한 인과 구조를 그려 교란변수(보정)·매개변수(총효과 추정 시 보정 금지)·collider(절대 보정/투입 금지)를 구분 [17] |
| **후문 기준(backdoor criterion)** | 어떤 변수집합 Z를 보정해야 `P(Y\|do(X))`를 식별하는지 알려줌. 잘못된 집합(하류·collider) 보정은 편향을 *생성* [17] |
| **RCT / target trial emulation** | 무작위화는 모든 후문 경로(`X ← Z`)를 끊음. WHI·CAST가 관찰 주장을 뒤집은 이유. 가능하면 기준점 |
| **민감도 분석 (E-value)** | 미측정 교란이 결과를 뒤집으려면 얼마나 강해야 하는지 정량화 |
| **음성 대조(negative controls)** | 영향을 줄 수 없는 노출/결과에서 "효과"가 나오면 잔여 교란·선택편향의 신호 |
| **"예측 ≠ 인과" 명시** | 높은 SHAP는 *이 분포에서 예측에 유용*하다는 증거이지 *개입하면 Y가 바뀐다*는 증거가 아님 — 실행 권고 전에 명시 |
| **도메인 전문가 검토** | Caruana의 천식 규칙은 임상의가 "생물학적으로 거꾸로"임을 알아 잡아냈다. 상위 변수의 부호·타당성을 전문가가 검토 |

## 8. 요약 체크리스트

모델 해석을 인과로 말하기 전에:

1. **이 변수가 결과를 *직접* 일으킬 수 있는 생물학적/물리적 경로가 있는가?** 없으면 대리·교란 의심.
2. **이 변수가 결과(질병)에 의해 *변하는* 양인가?** (혈압·체중·콜레스테롤·바이오마커) → 역인과 의심.
3. **이 변수가 "치료/약물/처방"인가?** → 적응증 교란. RCT/target trial 없이는 인과 금지.
4. **표본이 특정 기준(입원·검사·생존)으로 선택됐는가?** → collider/선택 편향. "보호 효과"가 의심스럽게 나오면 특히.
5. **이 변수가 지역·집단 평균인가?** → 생태학적 오류.
6. **변수를 더 넣었더니 다른 변수 부호가 뒤집혔는가?** → Simpson/교란/M-bias 점검.
7. **SHAP 변형이 interventional인가 conditional인가, 그 선택이 결론에 맞는가?**
8. **회귀표의 *모든* 계수를 인과로 읽고 있지 않은가?** → Table 2 fallacy.
